# Input

Here is how you use a normal input.

<Playground component={Input}>
  <ControlledField component={
    <Input placeholder="Placeholder..." />
  } />
</Playground>


You can also add a prefix and suffix.

export const style = {
  example: css`
    label: noreplace;
    width: 300px;
    > * {
      margin-bottom: 20px;
    }
  `,
};

export const Example = ({ children }) => {
  return (
    <div className={style.example}>
      {children}
    </div>
  );
}

<Playground>
  <Example>
    <Input placeholder="Something..." prefix="$" />
    <Input placeholder="Something..." suffix=".com" />
    <Input placeholder="Something..." prefix={
      <Select value="option" values={[{ label: 'Option', value: 'option' }]} placeholder="Select..." />
    } />
    <Input placeholder="Something else..." suffix={
      <Button withIcon="activity">Click me</Button>
    } />
  </Example>
</Playground>
