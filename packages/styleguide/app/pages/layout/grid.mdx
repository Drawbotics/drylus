# Grid
This component is used to organise elements on the page following a classic grid layout. To not be mistaken with the [`Flex`](/layout/flex) component which is used to organise elements in a vertical or horizontal row.

Grid only acts horizontally, creating a fixed number of columns, and a variable number of rows depending on the amount of children elements.

```jsx
import { Grid, GridItem } from '@drawbotics/react-drylus';
```

export const Component = () => {
  return (
    <div style={{ height: '50px', width: '100%', background: 'red', border: '1px solid orange' }} />
  );
};

Example:
<Playground component={Grid}>
  <Grid columns={4} hGutters={Sizes.DEFAULT} vGutters={Sizes.DEFAULT}>
    <GridItem>
      <Component />
    </GridItem>
    <GridItem span={2}>
      <Component />
    </GridItem>
    <GridItem>
      <Component />
    </GridItem>
    <GridItem>
      <Component />
    </GridItem>
    <GridItem>
      <Component />
    </GridItem>
  </Grid>
</Playground>

### Grid Item
The `GridItem` by default will take 1 column of space, and only 1 row. By changing the `span` prop you can define how many columns it will take.
