import {
  ListTile,
  RoundIcon,
  Categories,
  Icon,
  Margin,
  Sizes,
  Avatar,
  Text,
} from '@drawbotics/react-drylus';

import Playground from '~/components/Playground';


# List Tile
Inspired by the Flutter [component](https://api.flutter.dev/flutter/material/ListTile-class.html) of the same name.
```jsx
import { ListTile } from '@drawbotics/react-drylus';
```

__Example__
<Playground component={ListTile}>
  <ListTile
    title="Hello"
    subtitle="Im the subtitle"
    leading={<RoundIcon name="check" size={50} />}
    onClick={x=>x} />
</Playground>


The same component can be used for many things, such as combining an icon, or an avatar with text:

<Playground>
  <>
    <Margin size={{ bottom: Sizes.EXTRA_SMALL }}>
      <ListTile title="Destructive action" leading={<Icon name="x" category={Categories.DANGER} />} />
    </Margin>
    <Margin size={{ bottom: Sizes.EXTRA_LARGE }}>
      <ListTile title="Success action" leading={<Icon name="check" category={Categories.SUCCESS} />} />
    </Margin>
    <ListTile title="Mr Man Boy" leading={<Avatar text="mb" category={Categories.SUCCESS} />} />
  </>
</Playground>

You can use a custom Text component to change the look of the text for the `ListTile` `title` and `subtitle` properties:

<Playground>
  <ListTile title={<Text bold>Custom text</Text>} subtitle={<Text category={Categories.DANGER}>Same</Text>} leading={<RoundIcon name="check" size={50} />} />
</Playground>
