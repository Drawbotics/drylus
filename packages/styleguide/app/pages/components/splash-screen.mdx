# Splash Screen
Used when a whole page should be loading and the user shouldn't perform any action. For example when the website is loading, or when an order is being placed.

The component is shown when rendered, no props are required for that.

<PropsTable component={SplashScreen} />


### Demo
Press `s` on your keyboard to start the splash screen. Press it again to hide it.

export const Demo = () => {
  const [visible, toggle] = useState(false);
  useEffect(() => {
    const handlePressKey = (e) => {
      if (e.code === 'KeyS') {
        toggle(! visible);
      }
    };
    window.addEventListener('keydown', handlePressKey, false);
    return () => {
      window.removeEventListener('keydown', handlePressKey);
    };
  });
  return <SplashScreen visible={visible} />
};
