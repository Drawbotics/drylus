# Tooltip
It can be used to wrap any component/element and display a hint when hovered. The component is uncontrolled and uses portals to be rendered on top of the content.

<Playground component={Tooltip} enums={{ TooltipSides }}>
<Flex vSpacing={Sizes.LARGE}>
  <FlexItem>
    <Tooltip message="I am a tooltip">
      <Button category={Categories.BRAND}>Hover me</Button>
    </Tooltip>
  </FlexItem>
  <FlexItem>
    <Tooltip message="I am another tooltip">
      <Dot category={Categories.SUCCESS} />
    </Tooltip>
  </FlexItem>
</Flex>
</Playground>
