# Tooltip
It can be used to wrap any component/element and display a hint when hovered. The component is uncontrolled and uses portals to be rendered on top of the content.
```jsx
import { Tooltip, TooltipSides } from '@drawbotics/react-drylus';
```

__Example__
<Playground component={Tooltip} enums={{ TooltipSides }}>
<Flex>
  <FlexItem>
    <Margin size={{ right: Sizes.LARGE }}>
      <Tooltip message="I am a tooltip">
        <Button category={Categories.BRAND}>Hover me</Button>
      </Tooltip>
    </Margin>
  </FlexItem>
  <FlexItem>
    <Tooltip
      message={
        <div style={{ textAlign: 'left' }}>
          <Margin size={{ bottom: Sizes.EXTRA_SMALL }}>
            <ListTile leading={<Icon name="alert-triangle" category={Categories.DANGER} />} title={<Text inversed size={Sizes.SMALL}>I'm custom content</Text>} />
          </Margin>
          <ListTile leading={<Icon name="check-circle" category={Categories.SUCCESS} />} title={<Text inversed size={Sizes.SMALL}>So am I</Text>} />
        </div>
      }>
      <Dot category={Categories.SUCCESS} />
    </Tooltip>
  </FlexItem>
</Flex>
</Playground>
