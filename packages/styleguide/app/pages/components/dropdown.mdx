# Dropdown
A little floating menu used to perform actions. Can also contain some information, but should be kept at a minimum. If you see that the content of the dropdown is becoming excessive, use a [`Modal`](/components/modal) instead.

```jsx
import { Dropdown, DropdownOption, DropdownTitle, DropdownSeparator, DropdownSides } from '@drawbotics/react-drylus';
```
__Example__

<Playground component={Dropdown} enums={{ DropdownSides }}>
  <Dropdown trigger={
    <Button>I show the menu</Button>
  }>
    <DropdownOption text="First option" />
    <DropdownOption text="Second option" />
    <DropdownOption text="Third option" disabled />
    <DropdownSeparator />
    <DropdownOption text="Fourth option" />
  </Dropdown>
</Playground>
