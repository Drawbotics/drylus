# Panel

Usage
```js
import { Panel, PanelHeader, PanelBody, PanelSection, PanelFooter } from '@drawbotics/react-drylus';
```
Where `PanelBody` takes `PanelSection` as children. See example:

export const Content = () => {
  return (
    <div style={{ width: '100%', height: '200px', background: 'orange', padding: sv.defaultPadding }}>
      Some content goes here
    </div>
  );
};

<Playground component={Panel}>
  <Panel
    header={
      <PanelHeader>
        <Flex justify={FlexJustify.SPACE_BETWEEN}>
          <FlexItem flex>
            <Title size={4} noMargin>Header of the panel</Title>
          </FlexItem>
          <FlexItem>
            <Margin size={{ right: Sizes.SMALL }}>
              <Select
                placeholder="Filter 1..."
                options={[
                  { label: 'Option 1', value: '1' },
                  { label: 'Option 2', value: '2' },
                ]}
              />
            </Margin>
          </FlexItem>
          <FlexItem>
            <Select
              placeholder="Another action..."
              options={[
                { label: 'Option 1', value: '1' },
                { label: 'Option 2', value: '2' },
              ]}
            />
          </FlexItem>
        </Flex>
      </PanelHeader>
    }
    body={
      <PanelBody>
        <PanelSection title="Revenue">
          <Content />
        </PanelSection>
        <PanelSection title="Growth">
          <Content />
        </PanelSection>
      </PanelBody>
    }
    footer={
      <PanelFooter>
        <Flex justify={FlexJustify.END}>
          <FlexItem>
            <Margin size={{ right: Sizes.SMALL }}>
              <Button tier={Tiers.TERTIARY}>Tertiary action</Button>
            </Margin>
          </FlexItem>
          <FlexItem>
            <Margin size={{ right: Sizes.SMALL }}>
              <Button tier={Tiers.SECONDARY}>Secondary action</Button>
            </Margin>
          </FlexItem>
          <FlexItem>
            <Button category={Categories.BRAND}>Primary action</Button>
          </FlexItem>
        </Flex>
      </PanelFooter>
    } />
</Playground>

Both `PanelHeader` and `PanelFooter` don't have any styles, they are simply containers. You should build your own content, and normally we use that space for actions (on either or both sides, or that take the full width).

#### PanelHeader
`PanelBody` and `PanelFooter` take identical props.

### Common examples

Here is an example found in the Backoffice: the main content is a table, and we have some pagination at the bottom to navigate the different pages.

      <PanelBody noPadding>
