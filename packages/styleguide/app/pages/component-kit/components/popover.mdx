# Popover
It can be used to wrap any component/element and display a hint when _clicked_. The component is uncontrolled and uses portals to be rendered on top of the content.
```jsx
import { Popover } from '@drawbotics/react-drylus';
```

__Example__
<Playground component={Popover}>
  <Flex>
    <FlexItem>
      <Margin size={{ right: Size.LARGE }}>
        <Popover content="I am a popover">
          <Button category={Category.BRAND}>Click me</Button>
        </Popover>
      </Margin>
    </FlexItem>
    <FlexItem>
      <Popover
        content={
          <div style={{ textAlign: 'left' }}>
            <Margin size={{ bottom: Size.EXTRA_SMALL }}>
              <ListTile leading={<Icon name="alert-triangle" color={Color.RED} />} title={<Text size={Size.SMALL}>I'm custom content</Text>} />
            </Margin>
            <ListTile leading={<Icon name="check-circle" color={Color.GREEN} />} title={<Text size={Size.SMALL}>So am I</Text>} />
          </div>
        }>
        <Avatar color={Color.GREEN} text="PO" />
      </Popover>
    </FlexItem>
  </Flex>
</Playground>
