# Toggle
Similar use to a checkbox, but not used with forms. Normally used to indicate that something has been __"activated"__ rather than __"selected"__.
```jsx
import { Toggle } from '@drawbotics/react-drylus';
```

__Example__
<Playground component={Toggle}>
  <ControlledBoolField component={
    <Toggle />
  } />
</Playground>
