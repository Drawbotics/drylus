# Dropdown
A little floating menu used to perform actions. Can also contain some information, but should be kept at a minimum. If you see that the content of the dropdown is becoming excessive, use a [`Modal`](/component-kit/components/modal) instead.

```jsx
import { Dropdown, DropdownOption, DropdownTitle, DropdownSeparator, DropdownSides } from '@drawbotics/react-drylus';
```
__Example__

<Playground component={Dropdown} enums={{ DropdownSides }}>

  }>

    <DropdownOption text="Second option" />
    <DropdownOption text="Third option" disabled />

### Example with custom content
You can also put whatever you want in a dropdown, as long as it respects the design system
<Playground>
  <Dropdown trigger={<Avatar text="JD" />} side={DropdownSides.RIGHT}>
    <Margin size={{ top: Sizes.EXTRA_SMALL, right: Sizes.SMALL, left: Sizes.SMALL }}>
      <Text style={{ whiteSpace: 'nowrap' }}>John Doe</Text>
    </Margin>
    <Margin size={{ right: Sizes.SMALL, left: Sizes.SMALL }}>
      <Text tier={Tiers.SECONDARY} size={Sizes.SMALL}>Drawer</Text>
    </Margin>
    <DropdownSeparator />
    <DropdownOption text="Settings" />
    <DropdownOption text="Logout" />
  </Dropdown>
</Playground>
