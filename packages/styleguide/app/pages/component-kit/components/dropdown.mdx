# Dropdown
A little floating menu used to perform actions. Can also contain some information, but should be kept at a minimum. If you see that the content of the dropdown is becoming excessive, use a [`Modal`](/component-kit/components/modal) instead.

```jsx
import { Dropdown, DropdownOption, DropdownTitle, DropdownSeparator } from '@drawbotics/react-drylus';
```
__Example__

<Playground component={Dropdown}>

    <Button>I show the menu</Button>

### Example with links
Options in the dropdown can also be links, when using DropdownLink
<Playground>
  <Dropdown trigger={<Avatar text="JD" />} side={Position.RIGHT}>
    <DropdownLink text="Settings" href="" />
    <DropdownOption text="Logout" />
  </Dropdown>
</Playground>


### Example with custom content
You can also put whatever you want in a dropdown, as long as it respects the design system
<Playground>
  <Dropdown trigger={<Avatar text="JD" />} side={Position.RIGHT}>
    <Margin size={{ top: Size.EXTRA_SMALL, right: Size.SMALL, left: Size.SMALL }}>
      <Text style={{ whiteSpace: 'nowrap' }}>John Doe</Text>
    </Margin>
    <Margin size={{ right: Size.SMALL, left: Size.SMALL }}>
      <Text tier={Tier.SECONDARY} size={Size.SMALL}>Drawer</Text>
    </Margin>
    <DropdownSeparator />
    <DropdownOption text="Settings" />
    <DropdownOption text="Logout" />
  </Dropdown>
</Playground>
