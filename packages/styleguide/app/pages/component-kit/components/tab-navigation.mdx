# Tab Navigation

It is primarily used to change the context of a whole page, that is in contrast with `SegmentedControl` which only should modify part of a page's content.
```jsx
import { TabNavigation } from '@drawbotics/react-drylus';
```

__Example__

This is the most basic usage of a tab navigation:

<Playground component={TabNavigation}>
  <ControlledField component={
    <TabNavigation
      options={[{
        label: 'First choice',
        value: 'first',
      }, {
        label: 'Second choice',
        value: 'second',
      }, {
        label: 'Third',
        value: 'third',
        bullet: 4,
        disabled: true,
      }]} />
  } />
</Playground>

However it can also be set vertically, and each value may have bullets (similarly to `SegmentedControl`), and may also be disabled.

<Playground>
  <div style={{ maxWidth: 300 }}>
    <ControlledField component={
      <TabNavigation
        vertical
        options={[{
          label: 'First choice',
          value: 'first',
          bullet: 4,
        }, {
          label: 'Second choice',
          value: 'second',
          bullet: 10,
          disabled: true,
        }, {
          label: 'Third',
          value: 'third',
          bullet: 123,
        }, {
          label: 'Fourth',
          value: 'fourth',
          bullet: 40,
        }]} />
      } />
  </div>
</Playground>

Each option can also be given a `loading` value (useful when loading the data to determine the bullet number):
