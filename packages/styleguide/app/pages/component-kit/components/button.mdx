  Tier,
  Size,
  Category,
  Icon,

import Playground from '~/components/Playground';


# Button

Usage
```jsx
import { Button } from '@drawbotics/react-drylus';
```
__Example__
<Playground component={Button}>
  <Button>Button Text</Button>
</Playground>


You can also have buttons with just the icon, or just the spinner; for this don't pass any children. "No Children" buttons only support default and small size.
