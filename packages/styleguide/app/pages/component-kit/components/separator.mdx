# Separator
Simply a decorative component used to add a thin grey bar between contents in a page or other container. If you want to add spacing you should use the `Margin` and `Padding` components.
```jsx
import { Separator } from '@drawbotics/react-drylus';
```

__Example__
<Playground component={Separator}>
  <div style={{ height: 100 }}>
    <Separator />
  </div>
</Playground>
