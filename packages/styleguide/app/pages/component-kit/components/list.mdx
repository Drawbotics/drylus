# List
Used to display items with bullet points or numbers. `ordered` lists take over most of the items style, and make it all the same color, overriding `category` and `icon`.

```jsx
import { List, ListItem } from '@drawbotics/react-drylus';
```
__Example__

<Playground component={List}>
  <List>
    <ListItem>First item in the list</ListItem>
    <ListItem>Second item in the list</ListItem>
    <ListItem>Third item in the list</ListItem>
    <ListItem>Fourth item in the list</ListItem>
    <div></div>
  </List>
</Playground>
