# Tooltip
It can be used to wrap any component/element and display a hint when hovered. The component is uncontrolled and uses portals to be rendered on top of the content.
```jsx
import { Tooltip } from '@drawbotics/react-drylus';
```

__Example__
<Playground component={Tooltip}>
  <Flex>
    <FlexItem>
      <Margin size={{ right: Size.LARGE }}>
        <Tooltip content="I am a tooltip">
          <Button category={Category.BRAND}>Hover me</Button>
        </Tooltip>
      </Margin>
    </FlexItem>
    <FlexItem>
      <Tooltip
        content={
          <div style={{ textAlign: 'left' }}>
            <Margin size={{ bottom: Size.EXTRA_SMALL }}>
              <ListTile leading={<Icon name="alert-triangle" color={Color.RED} />} title={<Text inversed size={Size.SMALL}>I'm custom content</Text>} />
            </Margin>
            <ListTile leading={<Icon name="check-circle" color={Color.GREEN} />} title={<Text inversed size={Size.SMALL}>So am I</Text>} />
          </div>
        }>
        <Dot color={Color.GREEN} />
      </Tooltip>
    </FlexItem>
  </Flex>
</Playground>
