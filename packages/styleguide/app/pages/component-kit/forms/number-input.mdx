# Number Input
Used in forms to set a numerical value to a field. You can also type directly in the input but only positive and negative integers.
```jsx
import { NumberInput } from '@drawbotics/react-drylus';
```

__Example__
<Playground component={NumberInput}>
  <ControlledField initialValue="" style={{ width: 300 }} component={
    <NumberInput placeholder="Number of things" value="" />
  } />
</Playground>
