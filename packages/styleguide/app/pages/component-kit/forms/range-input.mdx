# Range Input
Similar to the native `input` of `range` type, but allows also more than 1 handle i.e. more than 1 value.
```jsx
import { RangeInput } from '@drawbotics/react-drylus';
```

__Example__
<Playground component={RangeInput}>
  <ControlledField style={{ maxWidth: 300, width: '100%' }} initialValue={0} component={
    <RangeInput step={1} onChange={x => x} max={100} min={0} value={0} />
  } />
</Playground>
