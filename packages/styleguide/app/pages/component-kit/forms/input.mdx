# Input
Usage
```jsx
import { Input } from '@drawbotics/react-drylus';
```

__Example__
<Playground component={Input}>
  <ControlledField style={{ maxWidth: 250 }} component={
    <Input placeholder="Placeholder..." onChange={x=>x} value="" />
  } />
</Playground>

The `type` property has been limited to a few possible from the HTML definition, since we have specific components for some types (see the rest of the Forms section). For each type you define, you can pass the native properties to customize it, e.g. for `type="email"` you may pass `pattern=".+@globex.com" size="30"`

If no `onChange` value is passed, the field is automatically set to `read-only`, which doesn't allow the value to be modified.

<Playground>
  <Input value="The value is locked" style={{ maxWidth: 300 }} />

You can also add a prefix and suffix.

export const style = {
  example: css`
    label: noreplace;
    width: 300px;
    > * {
      margin-bottom: 20px;
    }
  `,
};

export const Example = ({ children }) => {
  return (
    <div className={style.example}>
      {children}
    </div>
  );
}
