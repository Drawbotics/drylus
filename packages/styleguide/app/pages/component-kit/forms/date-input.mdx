import { DateInput, Align } from '@drawbotics/react-drylus';

import Playground from '~/components/Playground';
import { ControlledField } from '../utils';


# Date Input
Usage
```jsx
import { DateInput } from '@drawbotics/react-drylus';
```

__Example__
<Playground component={DateInput}>
  <div style={{ maxWidth: 300 }}>
    <ControlledField initialValue="" component={
      <DateInput value="" align={Align.RIGHT} placeholder="Pick a date..." />
    } />
  </div>
</Playground>

If you need to limit the range of dates you can use `maxDate` and/or `minDate` (see the [react-calendar](https://github.com/wojtekmaj/react-calendar#readme) documentation for the full list of options).

<Playground>
  <div style={{ maxWidth: 300 }}>
    <ControlledField initialValue="" component={
      <DateInput
        value=""
        placeholder="Pick a date..."
        maxDate={{ year: 2010, month: 4, day: 10 }}
        activeStartDate={{ year: 2010, month: 4, day: 10 }}
        minDate={{ year: 2010, month: 4, day: 1 }}
        onChange={x=>x} />
    } />
  </div>
</Playground>
