# Phone Number Input
Used in forms to set a phone number. Includes a select to change the number prefix with country flag emojis.
```jsx
import { PhoneNumberInput } from '@drawbotics/react-drylus';
```

__Note__
Because the phone number prefix is automatically applied to the input when the component loads, we can't easily check if the value is empty or not in the form control. Thus the component exposes a utility function `getTrailingPhoneNumber` which returns the number without the prefix.
