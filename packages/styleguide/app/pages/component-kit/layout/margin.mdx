# Margin
Low level component used to wrap other elements with margin.
```jsx
import { Margin } from '@drawbotics/react-drylus';
```

__Example__
<Playground component={Margin}>
  <div style={{ border: '1px solid red', display: 'inline-block' }}>
    <Margin>
      <div style={{ background: 'green', height: 100, width: 100 }} />
    </Margin>
  </div>
</Playground>

You can also pass an object to specify the margin on each side. You can see below that the two `Margin` components, when placed side by side, merge their margins. This is not the case with the `Padding` component. Inspect the code below to see the effect.

<Playground>
  <>
    <Margin size={{
      top: Sizes.LARGE,
      bottom: Sizes.LARGE,
      left: Sizes.EXTRA_SMALL,
    }}>
      <div style={{ background: 'green', height: 100, width: 100 }} />
    </Margin>
    <Margin size={{
      top: Sizes.LARGE,
      bottom: Sizes.EXTRA_SMALL,
      left: Sizes.DEFAULT,
    }}>
      <div style={{ background: 'green', height: 100, width: 100 }} />
    </Margin>
  </>
</Playground>
