# Padding
Low level component used to wrap other elements with padding.
```jsx
import { Padding } from '@drawbotics/react-drylus';
```

__Example__
<Playground component={Padding}>
  <div style={{ border: '1px solid red', display: 'inline-block' }}>
    <Padding>
      <div style={{ background: 'green', height: 100, width: 100 }} />
    </Padding>
  </div>
</Playground>

You can also pass an object to specify the padding on each side. You can pass an object containing each side definition (`left`, `right`, `top`, `bottom`) or each direction (`vertical`, `horizontal`). Each key can contain a value of `Size`.

As you can see, the paddings are not combined and each element keeps its padding as expected.

<Playground>
  <>
    <Padding
      size={{
        top: Size.LARGE,
        bottom: Size.EXTRA_SMALL,
        left: Size.DEFAULT,
      }}
      style={{ border: '1px solid red' }}>
      <div style={{ background: 'green', height: 100, width: 100 }} />
    </Padding>
    <Padding
      size={{
        top: Size.LARGE,
        bottom: Size.EXTRA_SMALL,
        left: Size.MASSIVE,
      }}
      style={{ border: '1px solid red' }}>
      <div style={{ background: 'green', height: 100, width: 100 }} />
    </Padding>
    <Padding
      size={{
        vertical: Size.SMALL,
        horizontal: Size.LARGE,
      }}
      style={{ border: '1px solid red' }}>
      <div style={{ background: 'green', height: 100, width: 100 }} />
    </Padding>
  </>
</Playground>
