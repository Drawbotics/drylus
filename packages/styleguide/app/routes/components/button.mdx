# Button

The following is an non-editable code example
```jsx
<button class="Drylus-Button__base Drylus-Button__danger Drylus-Button__small Drylus-Button__secondary">
  <button class="Drylus-Button__base">Button Text</button>
</button>
```

The following is the rendered component
<Playground component={Button}>
  <Button>Button Text</Button>
</Playground>


You can also have buttons with just the icon; for this don't pass any children. "Icon only" buttons only support default and small size. Icon only buttons have a lighter tone by default, illustrating they should be used for secondary actions.
<Playground>

  </div>
</Playground>
